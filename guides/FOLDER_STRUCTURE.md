## Quick Setup Steps:

1. **Create the assets folder** next to portfolio.html
2. **Add your button PNGs** (these create the navigation buttons)
3. **Add your artwork images** (these appear in the gallery)
4. **Open portfolio.html** in a browser to see your portfolio!

## File Naming Tips:

- Use lowercase letters
- Replace spaces with hyphens or underscores
- Examples: `sunset-painting.jpg`, `digital_portrait.png`
- Match the exact names you use in the HTML
